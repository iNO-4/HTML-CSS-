HTML/CSS_復習メモ

  • ＜li＞は＜ul＞タグで囲む
    ＜ul＞
     ＜li＞＜/li＞
     ＜li＞＜/li＞
    ＜/ul＞
  • ＜head＞要素はページに関する情報
    ＜body＞は実際に表示したい内容
  • font-family
    フォント名にスペースがある場合は、”” で囲う
    p {
     font-family: “sample text”
    }
  • ＜li＞要素に、list-style: none; をcssで指定すると黒点が消える
  • floatプロパティを用いて要素を横並びにできる
    li {
     float: left(right);
    }
  • paddingやmarginを指定する時は、スペース区切りの場合、
    「上」「右」「下」「左」の順で適用される。値を２つ指定した場合、「上下」「左右」の順に適用される
    .sample {
     padding: 10px(top) 10px(right) 10px(bottom) 10px(left);
    }
    .sample {
     padding: 10px(top+bottom) 10px(right+left);
    }
  • ＜span＞要素の前後に改行が入らず、基本的に横並びになる
  • 「ブロック要素」「インライン要素」
    前後で改行が入って、親要素の幅いっぱいに広がるのは「ブロック要素」(＜div＞や＜h1＞など)
    対して、改行されない要素は「インライン要素」(＜span＞や＜a＞など)
  • ＜img＞要素の使い方
    ＜img src="この中に入る"＞
  • ＜input＞＜textarea＞
    ＜input＞要素は１行のテキスト入力を受けるための要素(＜input＞要素は終了タグが不要な点に注意)
    ＜textarea＞要素は複数行のテキスト入力を受けるための要素
  • ボタンの作成
    ＜input＞要素にはtype属性を指定できる
    type属性にsubmitを指定するとボタンになる
    ＜input type="submit"＞...送信ボタン(デフォルト)
    valueでボタンに表示されるテキストを変更できる...＜input type="submit" value="保存"＞
